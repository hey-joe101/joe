#### position fixed 滚动条失效
不要设置高度!

```
    .fixed-content {  
        top: 0;  
        bottom:0;  
        position:fixed;  
        overflow-y:scroll;  
        overflow-x:hidden;  
    }
```

#### position: fixed受transform的影响

当使用了position:fixed 的元素，如果其祖先元素存在 transform 值不为 none ，那么该元素将相对于设定了 transform 的祖先元素进行定位。
